# ExX2Y2
[![hex.pm version](https://img.shields.io/hexpm/v/ex_x2y2.svg?style=flat)](https://hex.pm/packages/ex_x2y2)

X2Y2 API client for Elixir

## Installation

Add the `ex_x2y2` package to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_x2y2, "~> 0.0.1"}
  ]
end
```
